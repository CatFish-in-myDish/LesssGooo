Lesss Gooo is an AI-powered travel planning web application built as a front-end UI/UX project.
It features a full multi-page interface including a landing page, user authentication flow, an itinerary planner dashboard
, bookings manager, profile page, and settings — all styled with a modern dark theme. 
Users can simulate generating personalized travel itineraries by entering their destination, budget, travel style, and interests, 
and manage a full list of trips and bookings with view, edit, and delete functionality.
Built with vanilla HTML, CSS, and JavaScript, the app is fully responsive and uses localStorage for client-side state management.
